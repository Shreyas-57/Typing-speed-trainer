# PulseType Trainer

PulseType Trainer is a stylish typing speed practice app built with plain HTML, CSS, and JavaScript.

## Features

- Random paragraph practice
- 30, 60, and 120 second timer modes
- Live WPM, accuracy, and mistake tracking
- Character-by-character highlighting
- Weak key analysis
- Custom practice modes for weak keys, numbers, symbols, and code snippets
- Dark mode
- Local history saved in the browser

## Run locally

Open `index.html` in any modern browser.

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Push this folder to the repository.
3. In the repository settings, open `Pages`.
4. Set the source to deploy from the `main` branch and `/root`.
5. Save and wait for the public URL.
